# MovieLens Service

A REST API for querying and analyzing a movie dataset under load.

## Features
- Search movies by title
- Filter movies by genre
- Get movie details with rating stats
- Get top-rated movies
- Get genre statistics

## Stack
- FastAPI
- PostgreSQL
- Redis
- Docker
- k6

## Run locally
```bash
docker compose up --build
